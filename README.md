# visual_projects

Projects from the **Data Visualization** and **Working with APIs** sections of *Python Crash Course* by Eric Matthes.

## Projects

### Beginner Projects
- **Matplotlib Squares** — plotting squares with `mpl_squares.py` and `scatter_squares.py`
- **Random Walk** — simulating and visualizing random movement (`random_walk.py`, `rw_visual.py`)
- **Dice Rolls** — simulating dice rolls and visualizing distributions (`die.py`, `dice_visual.py`)

### A Bit of Analysis
- **Sitka Weather** — charting temperature highs and lows from real CSV data (`sitka_highs.py`, `sitka_highs_lows.py`)
- **Death Valley Weather** — analyzing temperature data from Death Valley (`death_valley_highs_lows.py`)
- **Earthquake Data** — exploring and mapping global earthquake data (`eq_explore_data.py`, `eq_world_map.py`)

### Working with APIs
- **GitHub Repos** — fetching and visualizing top Python repositories from GitHub API (`python_repos.py`)
- **Hacker News** — visualizing most-commented articles from Hacker News API (`hn_submissions.py`)

## Stack

- Python 3
- Matplotlib
- Plotly
- Requests
